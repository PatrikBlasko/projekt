# PHP Todo & Chat Aplikácia

Jednoduchá webová aplikácia na správu úloh a jednoduchý chat systém s PHP a MySQL.

## 📋 Popis Projektu

Aplikácia umožňuje:
- **Todo List**: Vytvorenie, úpravu, mazanie a označovanie úloh ako hotové
- **Chat**: Jednoduchý chat s možnosťou odosielania správ
- **Používatelia**: Základný systém používateľov s prihlasovaním
